JS thần công quyển 2 - làm việc với array phần 1

Lúc đi làm tại hạ làm việc với mảng nhiều vô kể, mà với mớ công pháp backend của kẻ này thì cũng phải tốn thời gian đọc tài liệu rồi mới xử lý được 1 số task, mất thời gian. Vậy nên tại hạ viết ra đây cho các đạo hữu tham khảo. Ở bài này mình mở ngay tab console của chrome ra code ví dụ luôn nghen.

Tạo, thêm, xóa phần tử của mảng

Để khai báo 1 mảng ta có thể làm như sau:

var myArray = [] // mảng trống
myArray = [1,2,3,4,'Kiên Đinh'];
console.log(myArray);
//In ra số phần tử của mảng
console.log(myArray.length);
// Lấy ở vị trí nào đó
console.log(myArray[4]);
// Kết quả là: 
// [1, 2, 3, 4, "Kiên Đinh"]
// 5
// Kiên Đinh

push và pop, shift và unshift

push sẽ thêm phần tử mới vào cuối mảng đó pop thì ngược lại, sẽ lấy ra phần tử cuối cùng của mảng đó Nếu làm việc ở đầu mảng thì sẽ dùng shift để lấy ra phần tử đầu tiên và unshift để thêm vào giá trị mới ở vị trí đầu tiên

myArray.push('ABC');
// Trả về số phần tử của myArray : 6
myArray.pop();
// Trả về phần tử cuối cùng của mảng đó: 'ABC', và remove luôn khỏi mảng

Chạy vòng lặp (Iteration)

Js có nhiều cách để duyệt hết phần tử của mảng ta có thể dùng vòng for, hoặc forEach như hầu hết các ngôn ngữ khác:

//Vòng lặp bình thường
for(var i =0;i<myArray.length;i++)
{
	console.log(myArray[i]);
}
//forEach . ở đây mình dùng arrow function luôn
myArray.forEach((element) => {
	console.log(element);
});
// Nếu viết theo kiểu đúng callback như hồi xưa sẽ như này:
myArray.forEach(function (element) {
	console.log(element);
});

Kết quả 3 đoạn code trên đều giống nhau, tuy nhiên nếu quen dùng arrow function thì dùng cách 2 vẫn phê nhất :3
